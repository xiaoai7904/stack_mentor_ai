逻辑简析

  • createRef: 核心工厂函数，通过 new RefImpl 初始化一个响应式引用。

  • RefImpl 类: 实现了 getter 和 setter。Getter 负责调用 trackRefValue 追踪依赖；Setter 负责值比对及 triggerRefValue 派发更新。

  • __v_isRef: 标记位，用于 Vue 内部判断一个对象是否为由 ref 创建的代理。

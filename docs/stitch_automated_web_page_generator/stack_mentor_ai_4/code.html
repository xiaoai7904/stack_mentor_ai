Description

effect 是 Vue 3 响应式系统的核心。它是收集依赖、并在依赖变更时触发更新的基本单位。它是 render 函数、computed 和 watch 的底层支柱。

Implementation

function effect(fn, options) {
  // create reactive effect
  const _effect = new ReactiveEffect(fn)
  // init effect immediately
  _effect.run()
}

watch

subject Overview

watch is a function used to observe reactive changes in Vue 3. It can monitor ref, reactive objects, or a getter function, and execute a callback function when the data changes.

code Core Usage

TypeScript 
watch(source, (newValue, oldValue, onCleanup) => {
  // callback logic
}, {
  immediate: false,
  deep: false,
  flush: 'pre'
})
psychology

Mental Model

Think of watch as a subscriber to a data stream. It keeps a record of the "old" value and compares it with the "new" one to decide if a reaction is needed.

warning

Common Pitfalls

Avoid deep watching large objects as it causes heavy performance overhead. Always clean up side effects in the onCleanup hook.
